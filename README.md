# Modified_Martini3_Fluorocarbon
This repository contains the supporting data and codes used in "Parameter Optimization for Accurate Coarse-Grained Modeling of Perfluorocarbon Phase Behavior"
(i) The optimized parameters are stored in M3O_optimized.csv
(ii) The codes and examples for mean curvature analysis of the nanoparticles are stored in the Curvature_Analysis folder.
